Изучи прокт в текщей папке.
Проект представляет собой разрабоотку расширения для браузера Chrome. Расширние используется как интеллектуальный помощник для анализа текущих открытых пользовталем страниц. Расширение видит в каждый момент ту же страницу, что и пользователь. Если пользователь пермещается по открытм вкладкам в браузере или открывает новую страницу, расгирение сразу получает новый контекст и готово ответить на вопросы пользователя в контексте страницы, которую пользоватлеь видит в данный момет. Если пользователю нужно получить дополнительную информацию, отсутсвующую на текщей странице, расширение подключает сервис поиска.
В качетсве интеллектуального помощника используется Gemini Flash 2.5
Для поиска в интернете используется сервис EXA.
API KEY для Gemini И EXA содержатся в файле .env.local
В настоящее время это приложение работоспособно и используется по назначению.

Для дальнейшего развития проекта предлагается использовать LangGraph.
На данном этапе нужно переписать приложение, чтобы управление процессом осуществлялось на основе LangGraph.
Предложикак текущую функциональность реализовать с использованием LanngGraph.
